Características de JS

  • Isomórfico: se puede ejecutar en las 3 capas de una aplicación

    De alto nivel

    Debilmente tipado: las variables no se necesitan ser restrigidas a un tipo de dato

    Multiparadigma: POO, Funcional, estrucuturada, etc.

    Case Sensitive: perro != Perro

    Interpretado y no compilado

    Dinámico

    No se necesitan puntos y comas al final de una línea
